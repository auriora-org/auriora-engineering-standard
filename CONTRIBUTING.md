# Contributing

The AURIORA Engineering Standard (AES) is a normative document maintained by the AURIORA maintainers. Changes to normative text are deliberate: every released version records its normative changes in the [CHANGELOG](./CHANGELOG.md) per [AES-GOV-011](./docs/08-decisions-and-governance.md#aes-gov-011-standard-change-record), and platform-wide decisions are recorded as [EDRs](./docs/08-decisions-and-governance.md).

## Feedback

If you find an ambiguity, contradiction, defect or gap, please [open an issue](https://github.com/auriora-org/auriora-engineering-standard/issues). Name the affected chapter or requirement identifier (for example `AES-IF-005`) and what you believe is wrong or unclear.

## Pull Requests

Pull requests are welcome for editorial fixes (typos, broken links, formatting). For normative changes — anything that alters a MUST/SHOULD/MAY obligation — please open an issue first so the change can be discussed and recorded properly; the maintainer decides and documents the outcome.
